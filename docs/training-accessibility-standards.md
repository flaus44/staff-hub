# Training accessibility standards

For co-design facilitators (ages 18–40, many with disability).

## Content

- Reading level: **Grade 6–8**
- Module unit: **3–5 minutes** max
- One idea per screen
- Legal scripts (Section 7): verbatim with plain-English summary underneath

## UI

- WCAG 2.2 AA; 44px touch targets
- One quiz question per step where possible
- Unlimited friendly retries; no timers
- Save and resume between blocks
- Captions + transcript on all video
- Status: icon + text, not colour alone

## Assessment

- Critical modules (distress, Section 7/8): 2 scenario questions, stored responses
- Low-risk modules: tick attestation only
